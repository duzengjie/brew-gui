# Brew GUI Icon Prompts

## Contact Sheet Prompt

Create a single contact sheet with 4 distinct square macOS app icon candidates for an app named Brew GUI, a native Homebrew package manager GUI for macOS.

Requirements:

- Rounded-square app icon
- Opaque background
- No black border or outline frame
- No text, letters, numbers, or watermark
- Centered bold symbol
- Readable at 32px
- Youthful, modern, friendly, polished
- Suitable for macOS Dock and web favicon

Visual themes:

- Playful homebrew cup with package cube
- Colorful terminal window blended with a tea or coffee cup
- Clean package box with subtle steam
- Abstract brew droplet plus command sparkle

Palette:

- Mint
- Coral
- Warm cream
- Sky blue
- Soft amber
- Gentle gradients and soft dimensional lighting

Avoid:

- Dark-heavy palette
- Black strokes
- Realistic beer logos
- Apple or Homebrew trademarks
