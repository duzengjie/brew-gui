# Brew GUI Icon Choices

![Brew GUI icon contact sheet](contact-sheet.png)

## Options

1. Mint cup and package cube  
   Friendly, clear at small sizes, and strongly connected to both Homebrew and package management.

2. Coral terminal and cup  
   Most directly communicates "GUI for command-line tools"; energetic and young without feeling toy-like.

3. Amber package box with steam  
   Clean package-management metaphor, but slightly less distinctive as a Brew-specific app icon.

4. Sky-blue brew droplet and prompt mark  
   Fresh and abstract; good if the app should feel lighter and less literal.

## Recommendation

Option 2 is the strongest product icon direction because it combines terminal control, macOS-style color, and the brew cup metaphor in one readable shape. Option 1 is the best softer alternative if you want a warmer utility-app feel.

## Selected

Option 2 was selected and exported to:

- `AppIcon.icns`
- `AppIcon.iconset/`
- `../../Sources/BrewGUI/Resources/AppIcon.icns`

The final app icon was then scaled to 88% on a transparent 1024x1024 canvas so it appears smaller in the macOS Dock without adding a white border.
